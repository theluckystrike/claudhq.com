# ClaudHQ

> Free Claude prompt library with 30+ ready-to-use templates for coding, writing, analysis, and business workflows.

**[Use ClaudHQ live](https://claudhq.com/)**

ClaudHQ is a curated library of 30+ tested prompt templates designed for Anthropic's Claude. Browse prompts organized by category (coding, writing, analysis, creative, business, education), search and filter in real time, copy with one click, and save favorites locally. Every prompt includes token estimates and model recommendations for Claude Opus, Sonnet, and Haiku.

## Features

- 30+ tested, ready-to-use prompt templates for Claude Opus, Sonnet, and Haiku
- Six prompt categories: coding, writing, analysis, creative, business, education
- Real-time search engine filtering by title, description, and tags
- One-click copy-to-clipboard for any prompt
- Token count estimates and model recommendations per prompt
- Local favorites system stored in browser (no account needed)
- Prompt engineering best practices baked into every template
- Works with Claude web app, API, and Claude Code CLI
- 100% client-side — no data leaves your browser
- MIT licensed
- No signup, no tracking

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript
- No build step
- No external dependencies (except Google Fonts on some pages)
- Hosted on GitHub Pages with Cloudflare CDN
