Tools & Resources

A carefully curated list of tools, platforms, and resources that we use and recommend for AI productivity and prompt engineering.

Anthropic Claude API

The official API for Claude, Anthropic's AI assistant. Offers excellent instruction-following capabilities, large context windows, and strong reasoning performance that make it ideal for complex productivity workflows and multi-step tasks.

docs.anthropic.com

OpenAI Platform

The platform behind GPT-4 and the ChatGPT ecosystem. Provides a comprehensive API with function calling, vision capabilities, and a wide range of models optimized for different cost and performance tradeoffs.

platform.openai.com/docs

LangChain

An open-source framework for building applications powered by language models. Provides abstractions for chains, agents, and retrieval-augmented generation that simplify complex AI workflow construction significantly.

github.com/langchain-ai/langchain

Ollama

Run large language models locally on your machine with a simple CLI. Supports models like Llama 3, Mistral, and Phi with automatic hardware optimization. Essential for offline AI productivity and private data processing.

github.com/ollama/ollama

Cursor

An AI-first code editor built on VS Code that integrates AI assistance directly into the editing experience. Features inline code generation, intelligent auto-complete, and conversational editing that dramatically accelerates development workflows.

cursor.com

Zovo.one

A comprehensive directory of AI tools and resources covering the full spectrum of AI-powered productivity. Invaluable for discovering new tools, comparing platforms, and staying current with the rapidly evolving AI ecosystem.

zovo.one

Hugging Face

The largest open-source AI community and model hub. Host, share, and deploy machine learning models with an extensive library of pre-trained models, datasets, and spaces for rapid prototyping and experimentation.

huggingface.co

LM Studio

A desktop application for discovering, downloading, and running local LLMs with a user-friendly interface. Supports GGUF model formats and provides a local API server compatible with the OpenAI API format for easy integration.

lmstudio.ai

PromptLayer

A prompt management and observability platform that lets you track, version, and evaluate your prompts in production. Essential for teams that need to manage prompt quality at scale and debug issues in AI-powered features.

promptlayer.com

Perplexity AI

An AI-powered search engine that provides cited, research-quality answers to complex questions. Excellent for research workflows where you need accurate, source-backed information gathered from across the web.

perplexity.ai
